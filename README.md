### Using HTTPS in Development

#### Windows (cmd.exe)

```cmd
set HTTPS=true&&npm start
```

#### Windows (Powershell)

```Powershell
($env:HTTPS = $true) -and (npm start)
```

#### Linux, macOS (Bash)

```bash
HTTPS=true npm start
```
